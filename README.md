# daily-checklist-design
Visual daily checklist diagram with Figma design and AI prompt templates for productivity tracking


## Design Files

- **Figma Design**: [Daily-Checklist-Tonight](https://www.figma.com/design/1W43bUYcgsuHKRjE2TiCZ5/Daily-Checklist-Tonight)

## Daily Checklist Overview

A visual reminder and task tracker for nightly productivity sessions. The checklist includes:

### Tonight's Tasks (1-2 Hours)
1. ☐ Design 2 sticker characters
2. ☐ Save prompt templates
3. ☐ Stop installing tools

## Design Prompt

```
CREATE A CLEAN, MINIMALIST DAILY CHECKLIST DIAGRAM:

- Bold title: "TONIGHT (1-2 HOURS)" at the top
- Three task items with checkbox symbols (☐):
  1) Design 2 sticker characters
  2) Save prompt templates
  3) Stop installing tools
- Layout: Simple, organized, clear typography
- Color Scheme: Dark background with bright accents
- Style: Flat design, modern, professional
- Purpose: Visual reminder and task tracker for daily productivity
```

## Features

- ✨ Minimalist design aesthetic
- 🎯 Clear task organization
- ⏱️ Time-boxed workflow (1-2 hours)
- 📝 AI-generated design prompt template
- 🎨 Customizable color scheme

## Usage

This design serves as:
1. A daily visual reminder for focused work sessions
2. A template for creating similar productivity checklists
3. A prompt engineering example for AI-generated design specifications
